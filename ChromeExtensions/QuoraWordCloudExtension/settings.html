Help
  1. For answers with (more...) option, click on more to see the 'Get Word Cloud' option.
  2. Ensure that your background color and font colors are mutually exclusive.
  3. If some words are not appearing try increasing the size of the image.
  4. If you want to see a lot more words in the cloud decrease both maximum and minimum font size.
  5. If there is a lot of empty space in your word cloud try reducing the size of your image.
  6. Exclude common words will exclude words like 'if', 'a', 'an', 'and', etc, and is 'true' by default. If you want them to persist mark it false.
  7. Exclude words will allow you to exclude words from the answer of your liking. You can use this to remove any stray HTML tags that might come in the cloud.
  8. This tool does not work well if the answer is small or has too many hyperlinks/images :).
  9. This version of word cloud generator does not support Quora Posts.
  10. If the 'Get Word Cloud' button isn't visible try clicking on the answer text or reload the page. This should fix it
